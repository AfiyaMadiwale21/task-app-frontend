# Task Manager Frontend

This is the frontend of the Task Management Web App. It is built with React.js and Tailwind CSS, consuming the backend APIs for user authentication and task management.

---

## **Tech Stack**

- **React.js** – Frontend framework
- **Tailwind CSS** – Styling
- **React Router** – Client-side routing
- **Fetch API** – Communication with backend
- **Vercel** – Deployment of frontend
- **Render** – Backend deployment
- **MongoDB Atlas** – Database

---

## **Features**

- User registration and login
- Dashboard to add, edit, delete, and mark tasks as done
- Responsive and clean UI
- Task prioritization and due dates
